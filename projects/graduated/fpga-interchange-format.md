# Project application: FPGA Interchange format

## Application

### Sandbox application

* Project name: FPGA Interchange format
* Project repo(s): https://github.com/SymbiFlow/fpga-interchange-schema
* Brief summary of the project: FPGA Interchange is a Vendor agnostic FPGA devices and designs description. It enables interoperability between different FPGA tools.
* Project's open source license: Apache-2.0
* Link to issue tracker: https://github.com/SymbiFlow/fpga-interchange-schema/issues
* Link to website: https://fpga-interchange-schema.readthedocs.io/
* Links to social media accounts: N/A
* Who uses this project, and at what scale: Google, Antmicro, Xilinx, other users in the open source community
* Why does the project want to join CHIPS Alliance: To establish a wider community around open source FPGA tooling and enable research on different synthesis and place & route algorithms and strategies.
